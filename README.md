
# 📘 CSS Interview Guide

A complete **CSS Interview Questions reference** with bilingual answers (English & Arabic), built with **HTML**, **CSS**, and **JavaScript**.

---

## 🚀 Live Demo

🔗 [View Live](https://amanyhisham.github.io/CSS_Intervew_Question/)

---

## ✨ Features

- 📚 Organized CSS interview questions by topic (e.g. Box Model, Flexbox, etc.)
- 🔢 Numbered questions with expandable answers
- 🌐 Bilingual answers — English and Arabic
- 📂 Collapsible sections per topic
- 🔍 Easy to navigate with a menu icon
- 🎨 Clean and minimal UI

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
